Come funziona Il Tuo Agente Team Inizia
ENUKIT

Da zero a un sistema di apprendimento in cinque minuti

Il tuo primo agente parte intelligente. Diventa solo più intelligente.

Cinque passaggi. Nessun file di configurazione da scrivere. Nessuna chiave API richiesta.

1

Installa

Un solo pacchetto global npm. Tutto qui.

$ npm install -g soleri
2

Crea il tuo agente

Dagli un nome e connetti una cassaforte. Soleri configura i domini, installa conoscenze iniziali e scansiona il tuo progetto.

$ soleri create my-agent ✓ Configurazione agente creata ~/.soleri/agents/my-agent/ ✓ Cassaforte connessa conoscenze iniziali: 34 schemi ✓ Progetto scansionato React + TypeScript rilevato ✓ Acquisizione automatica 12 schemi di base di codice ✓ Cassaforte pronta 46 voci, vettorizzate Agente "my-agent" è pronto.
3

Alimenta con conoscenze

Aggiungi conoscenze alla cassaforte del tuo agente. Installa pacchetti della comunità, connetti una cassaforte del team o lascia che il cervello catturi schemi dal tuo lavoro.

# Aggiungi un pacchetto di conoscenze della comunità $ soleri packs install community/react-patterns ✓ react-patterns installato v0.3.0 ✓ 28 schemi aggiunti alla cassaforte ✓ Vettorizzato e collegato al grafico # Il cervello cattura schemi dal tuo lavoro ! Hai sostituito bg-blue-500 con bg-primary 3 volte. Catturare come schema? [y/n] ✓ Catturato: enforcement-token-semantico
4

Estendi

Connetti la cassaforte condivisa del tuo team o aggiungi più pacchetti di conoscenze. Le conoscenze si accumulano.

# Connetti la cassaforte condivisa del tuo team $ soleri vault connect git@github.com:team/vault.git ✓ Cassaforte del team connessa 142 voci ✓ Priorità di ricerca: agente → progetto → team # Controlla lo stato della cassaforte $ soleri vault status Voci: 216 vettorizzate, collegate al grafico Domini: 3 frontend, backend, trasversali Grafico: 412 archi
5

Controllo di integrità

Esegui doctor per verificare che tutto sia connesso e aggiornato.

$ soleri doctor ✓ Motore v1.2.0 aggiornato ✓ domain-design v0.8.0 aggiornato ✓ domain-review v0.5.0 aggiornato ✓ my-agent configurazione ok ✓ Cassaforte 216 voci, sana ✓ Cervello tracciamento abilitato ✓ Cassaforte del team connessa, sincronizzata

Continua

Come funziona Cassaforte, cervello e architettura di trasporto spiegati. Il Tuo Agente Crea, configura e sviluppa le conoscenze del tuo agente. Team e Ops Cassaforti condivise, pacchetti di conoscenze e bot Telegram.

## 🧰 How to use soleri

Once soleri is running, you can start building your AI assistant.

- Use the simple menus to create new "personas." Personas are profiles for your assistant to manage different tasks.

- Add information or notes that you want the assistant to remember. soleri will organize these in a vault, a secure place where your data stays safe.

- Use the memory features to teach your assistant new things or correct it when it makes mistakes.

- You can interact with the assistant through a chat-like window. Type questions or commands in plain English.

- The assistant learns from each interaction, improving responses over time.

## ❓ Troubleshooting common issues

- **Installer won’t run**

  Right-click the installer file and select "Run as administrator." If the error persists, check if your Windows version meets requirements.

- **Can't find soleri after installation**

  Look in the Start menu under "soleri." You can also search for "soleri" in the Windows search bar.

- **Assistant does not respond**

  Check your internet connection. If offline, make sure you completed the initial setup with internet access first.

- **Application crashes or freezes**

  Restart your computer and try opening soleri again. If problems continue, uninstall and reinstall the program.
